Onboarding

Mulai dari file, lanjut ke insight, lalu simpan dashboard.

1

Masukkan data ke chat

Drag file langsung ke area chat, atau klik tombol `+` untuk upload dataset dari perangkat Anda.

2

Minta agent menganalisis

Tanya dengan bahasa biasa seperti insight minggu ini, tren penjualan, atau anomali yang perlu dicek.

3

Generate dashboard

Biarkan agent menyusun widget dan halaman dashboard berdasarkan kekuatan data yang benar-benar tersedia.

4

Edit dan simpan

Sesuaikan widget, rapikan layout, lalu simpan dashboard untuk dipakai lagi di sesi berikutnya.
